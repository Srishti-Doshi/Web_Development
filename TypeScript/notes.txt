typescript Setup

    Setting up a TypeScript project 
        => create a ts file

    Configuring tsconfig.json 
        => in tsconfig file we do settings like (var a; not used variables, function that returns null etc) are to treat as erors
        => run "tsc --init" or "npx tsc --init" to get tsconfig file

    Compiling TypeScript
        => run "tsc" or "tsc filename.ts"
        => run "tsc --watch" for continuation compilation of code as writing and saving

    primitives and reference
        a = 12

        b = a

        a => [1,2,3,4]

        [] {} () => type => reference => any change leads to change in parent

        no brackets => primitive

        12      => primtive
        Prince  => primtive
        true    => primtive
        []      => reference
        {}      => reference
        "apple" => primtive
        false   => primtive
        12.34   => primtive
        ()      => reference
        var a = true => primitive
        var [2,4]    => reference

        primitive => can be copied directly
        reference => cannot be copied directly, other ways

        var a = [1,2,3,4]
        var b = a => here b does not copy a , it has a reference of a
