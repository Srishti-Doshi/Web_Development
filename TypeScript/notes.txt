typescript Setup

    Setting up a TypeScript project 
        => create a ts file

    Configuring tsconfig.json 
        => in tsconfig file we do settings like (var a; not used variables, function that returns null etc) are to treat as erors
        => run "tsc --init" or "npx tsc --init" to get tsconfig file

    Compiling TypeScript
        => run "tsc" or "tsc filename.ts"
        => run "tsc --watch" for continuation compilation of code as writing and saving
